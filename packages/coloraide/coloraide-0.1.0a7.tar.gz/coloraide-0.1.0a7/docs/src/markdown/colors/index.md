### Supported Colors

### HSV

[Learn about HSV](https://en.wikipedia.org/wiki/HSL_and_HSV)

**White Point**
:   D65

**Coordinates**
: 
    Name       | Range
    ---------- | -----
    hue        | 0 - 360
    saturation | 0 - 100
    value      | 0 - 100

**Supported Inputs**
: 
    ```pycon3
    >>> Color('hsv', [0, 0, 100])
    color(hsv 0 0 100 / 1)
    >>> Color('color(hsv 0 0 100)')
    color(hsv 0 0 100 / 1)
    >>> Color('color(hsv 0 0% 100%)')
    color(hsv 0 0 100 / 1)
    ```
