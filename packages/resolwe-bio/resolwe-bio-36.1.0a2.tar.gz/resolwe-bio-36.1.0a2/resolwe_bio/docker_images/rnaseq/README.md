# Docker image for RNA-Seq processes

The image is based on the [`docker.io/resolwebio/common`](
https://hub.docker.com/r/resolwebio/common/) image, derived from the
`ubuntu-18.04` version of [`docker.io/resolwebio/base`](
https://hub.docker.com/r/resolwebio/base/) image.

Included bioinformatics tools:
------------------------------
* BBMap (v37.90)
* Cufflinks (v2.2.1)
* Cutadapt (v2.4)
* DESeq2 (v1.24.0)
* DEXSeq (v1.30.0)
* dexseq_prepare_annotation2 (v1.0.0)
* edgeR (v3.26.6)
* gotea (v0.0.4)
* genometools (1.5.9)
* igvtools (v2.3.98)
* jbrowse (v1.16.6)
* loadSubread (0.0.1)
* regtools (0.5.0)
* rnanorm (1.3.0)
* RSEM (v1.4.0)
* rtracklayer (v1.44.2)
* Salmon (1.2.1)
* STAR (v2.7.0f)
* Trimmomatic (v0.36)
* tximport (v1.12.3)
* shRNAde (v1.0)
* UMI-tools (v1.0.0)
* Trim Galore (v0.6.6)
